# VBF Syndic Pro+

**Professional Co-Property Management Software**

VBF Syndic Pro+ is a comprehensive desktop application for managing co-property (syndic) operations — budgets, charges, payments, owner statements, audits, and legal compliance.

## ✨ Key Features

- **Budget Management** — Forecasts, actuals, variance analysis
- **Charge Distribution** — Sharing keys (tantièmes), per-unit breakdowns
- **Payment Tracking** — Record & reconcile owner payments
- **Owner Statements** — Individual relevés with detailed charge breakdown
- **Bank Import** — Import bank statements (Excel/CSV)
- **Bilan Comptable** — Full accounting balance sheet
- **Audit Engine** — Automated anomaly detection
- **Water Metering** — Individual consumption tracking
- **Document Management** — Store and organize syndic documents
- **Meeting Minutes (PV)** — Generate and export AG reports
- **Owner Portal Export** — Self-contained HTML portal for owners
- **Multi-language** — French & English interface
- **Offline-first** — All data stays on your machine, no cloud required

## 🔒 Security

- Local-only data storage (no cloud dependency)
- Content Security Policy (CSP) enforced
- Encrypted license validation
- Built with [Tauri](https://tauri.app) for minimal attack surface
